# MNR Partners

## Installation
Utiliser le script `install.sh` pour installer le projet.
```bash
sh ./scripts/install.sh
```

## Deployment
Recuperer les dernieres modifications du repository.
```bash
git pull
```

Puis utiliser le script `install.sh` pour installer le projet.
```bash
sh ./scripts/install.sh
```

## Refresh
En cas de modification de style/configuration, utiliser le script `refresh.sh` pour rafraichir le projet et les dependances.
```bash
sh ./scripts/refresh.sh
```
